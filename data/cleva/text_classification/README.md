## Source

- Data of `news` are sampled from FewCLUE [(Xu et al., 2021)](https://arxiv.org/abs/2107.07498).
- Data of `humor` are sampled from [SPR](https://github.com/DUTIR-Emotion-Group/CCL2020-Humor-Computation).
